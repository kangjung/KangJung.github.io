자주 묻는 질문
메일 파일이 서버에 저장되나요?
아니요. 파일 선택과 분석은 브라우저 내부에서만 이루어지며 메일 내용은 이 사이트로 전송되지 않습니다.
어떤 EML 파일을 열 수 있나요?
일반 텍스트, HTML, multipart 형식과 Base64·Quoted-Printable로 인코딩된 대부분의 표준 EML 파일을 지원합니다.
외부 이미지와 스크립트를 볼 수 있나요?
기본 안전 모드에서는 차단합니다. 보기 보안 메뉴에서 사용자가 허용할 수 있지만, 신뢰하는 메일에서만 사용하는 것을 권장합니다.
